# Protótipo de Jogo Estilo Minecraft com Ursina Engine
### Projeto experimental apenas

Este projeto é um protótipo simples de um jogo sandbox 3D inspirado em Minecraft, desenvolvido em Python utilizando a Ursina Engine. O objetivo foi explorar conceitos básicos de desenvolvimento de jogos, como movimentação em primeira pessoa, interação com objetos do cenário e manipulação dinâmica do ambiente.

O jogador pode se movimentar livremente pelo mundo utilizando um controlador em primeira pessoa e interagir com blocos através do mouse. Novos blocos podem ser adicionados ao cenário, enquanto blocos existentes podem ser removidos, simulando uma mecânica básica de construção e destruição presente em jogos do gênero sandbox.

## Tecnologias Utilizadas

* Python 3
* Ursina Engine

## Funcionalidades

* Movimentação em primeira pessoa
* Mundo composto por blocos 3D
* Adição de blocos com o botão esquerdo do mouse
* Remoção de blocos com o botão direito do mouse
* Sistema simples de interação utilizando raycasting
* Ambiente 3D renderizado em tempo real

## Objetivo do Projeto

Este projeto foi desenvolvido como um experimento para estudar os fundamentos da criação de jogos 3D em Python. Ele serve como base para futuras implementações, como geração procedural de terrenos, inventário, diferentes tipos de blocos e sistemas mais avançados de construção.

> Projeto desenvolvido para fins de aprendizado e exploração da Ursina Engine.
